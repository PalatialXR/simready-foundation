## Omniverse

To comply with this capability when creating graspable vectors in Omniverse, see:

- [Omniverse Physics Graspable Vectors](https://docs.omniverse.nvidia.com/extensions/latest/ext_physics/graspable-vectors.html)

## Manual Creation

Graspable vectors can be manually created using USD BasisCurves or mesh primitives:

- **BasisCurves**: Recommended approach for creating grasp vectors as they are viewable in most runtimes
- **Mesh**: Alternative approach, but visibility depends on runtime support

## URDF

You can use the URDF extension to import URDF files with graspable vectors into Omniverse:

- [URDF Importer Extension](https://github.com/isaac-sim/urdf-importer-extension)
