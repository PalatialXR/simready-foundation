# Thumbnail Exist

The robot interface asset file should contain a thumbnail. The thumbnail should be representative of the robot.

## Example

For a robot asset located at `Robots/Manufacturer/Robot_Name/Robot.usd`, the thumbnail should be at `Robots/Manufacturer/Robot_Name/.thumbs/256x256/Robot.png`.

## Why

The thumbnail should provide an accurate preview for the robot

## How to comply

- .thumbs/256x256/{robot name}.png should exist
