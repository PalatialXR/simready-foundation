# Requirements

## Summary

**Isaac Sim Composition** ensures assets are structured with optimal performance and proper USD composition patterns for Isaac Sim environments.

- [USD Kind and Model Hierarchy](https://openusd.org/dev/api/usd_page_front.html#usd_kind_and_model_hierarchy)
- [UsdGeom Xformable](https://openusd.org/dev/api/class_usd_geom_xformable.html)
